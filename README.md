#RobotFramework-API-Demo

## Requirements

- Python 3.7 or greater

# Build project
In order to build the project execute the following command in cmd or terminal

for Windows:

```
build.bat
```

for mac:
```
sh build.sh
```

# Execute testcases


In order to execute testcases the following command in cmd or terminal

```
RunUITests.bat
```
